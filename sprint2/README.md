# Debounce

Node-typescript App project with 2 interfaces: cli & react, and tested with jest


## Installation 

```
cd sprint2/debounce
npm i
cd src/front
npm i
```

## Usage React Interface
```
cd sprint2/debounce/src/front
npm run dev
```

## Usage Cli Interface
```
cd sprint2/debounce/src
npx tsc --jsx "react"
```
```bash
node ../dist/cli.js "your debounce text"
```

## Testing
```
cd cli
npx jest
```
